# 🧹 Preparación de Datos con Python | E-commerce Analytics

## 📋 Descripción del proyecto
Este proyecto consiste en el desarrollo de un flujo de trabajo integral para la obtención, limpieza, transformación y estructuración de datos provenientes de múltiples fuentes, incluyendo archivos CSV, Excel y tablas web.

El objetivo fue garantizar la calidad del dataset final mediante técnicas de tratamiento de valores nulos, detección de outliers, eliminación de duplicados y transformación avanzada de variables, dejando los datos listos para análisis exploratorio, reporting y futuros modelos predictivos.

## 🎯 Objetivo
Construir un pipeline de preparación de datos automatizado y eficiente utilizando Python, NumPy y Pandas, orientado a generar un dataset limpio, confiable y estructurado para la toma de decisiones.

## 🛠️ Tecnologías y librerías utilizadas
- **Python**
- **NumPy**
- **Pandas**
- **Visual Studio Code / Script Python**
- **CSV / Excel / Web scraping con `read_html()`**

## ⚙️ Proceso realizado

### 1. Generación y carga de datos
- Creación de datos ficticios con NumPy
- Importación desde CSV
- Lectura de archivos Excel
- Extracción de tablas desde páginas web

### 2. Limpieza de datos
- Identificación y tratamiento de valores nulos
- Eliminación de registros duplicados
- Corrección de formatos inconsistentes
- Transformación de tipos de datos

### 3. Detección y tratamiento de outliers
- Aplicación de técnicas estadísticas:
  - **IQR**
  - **Z-score**
- Evaluación del impacto de valores atípicos

### 4. Data Wrangling
- Creación de nuevas variables
- Transformación de columnas
- Aplicación de `apply()`, `map()` y `lambda`
- Normalización y discretización de datos

### 5. Agrupamiento y pivoteo
- Uso de `groupby()`
- Reestructuración con `pivot()` y `melt()`
- Integración de múltiples fuentes con `merge()` y `concat()`

### 6. Exportación final
- Dataset final en **CSV**
- Dataset final en **Excel**

## 💡 Habilidades demostradas
- Limpieza y preparación de datos
- Data wrangling
- Integración de múltiples fuentes
- Detección de outliers
- Manipulación avanzada con Pandas
- Automatización de flujos de trabajo
- Preparación de datos para análisis y machine learning
